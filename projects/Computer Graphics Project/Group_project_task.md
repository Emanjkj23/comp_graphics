For this exercise, you will work in groups to design and implement an interactive 3D simulation environment using basic graphics primitives. 
```Key Concepts
The project must demonstrate key concepts from the course, including 
1. Modelling (building multiple objects from simple shapes), 
2. All transformations as discussed in class, 
3. Viewing and projection (camera movement with both perspective and orthographic views), 
4. Lighting and shading (ambient, diffuse, and specular effects),
5. Colour and blending, 
6. Event handling (keyboard and mouse interaction) and 
animation generation concepts. 
```
The scene should be meaningful (a real-world environment setup) and allow users to interact with objects and navigate the environment in real time.
To ensure originality and understanding, you are expected to structure your code into logical organized modules that build the scene and easy to explain and follow. During presentation, all group members will be expected to demonstrate understanding of all features and explain the underlying concepts. Emphasis will be placed not only on functionality and visual output, but also on correct application of graphics principles, clarity of implementation, and the ability to justify design and transformation choices.
